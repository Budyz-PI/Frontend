Getting Started

This project was bootstrapped with Create React App.

Scripts

Start the app in development:
npm start
Opens http://localhost:3000 and reloads on changes.

Run tests:
npm test
Launches the test runner in interactive watch mode.

Build for production:
npm run build
Bundles the app into the build folder for deployment.
